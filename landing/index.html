Uses Google's Gemini 2.0 Flash for fast, accurate corrections. Free tier available with your own API key.

Alternative provider with access to multiple models. Configure as fallback or primary provider.

PRIVACY

Text is sent only to your configured AI provider. API keys stored in macOS Keychain. No analytics.
